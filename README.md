# CWM

一款基于 **Godot 4.6** 与 **C# (.NET 8)** 的 2D 程序化世界沙盒游戏：大型地块世界、生态演替、农耕、昼夜与季节、天气、野生动物与村民 AI，并支持 **局域网多人** 联机。

## 环境要求

- [Godot 4.6](https://godotengine.org/)（带 **.NET** / C# 支持）
- [.NET 8 SDK](https://dotnet.microsoft.com/download)

首次用编辑器打开工程时会自动还原 NuGet / 生成 C# 工程；也可在仓库根目录执行：

```bash
dotnet build
```

在 Godot 中打开本目录，将 **主场景** 设为 `res://Scenes/Main.tscn`（`project.godot` 已配置），按 **F5** 运行。

## 主要玩法

| 方向 | 说明 |
|------|------|
| **世界** | 程序化生成大地图（种子、尺寸可调），主机权威地形与生态；支持世界导出/导入。 |
| **生态** | 演替、植被与气候相关模拟；主机可调演替加速等（部分选项仅主机）。 |
| **农耕** | 耕地、作物生长、杂草、灌溉与收获；收获进入粮仓，可与村民互动消耗粮食。 |
| **村民** | 游荡、建村、自主耕种与清理林地；每位玩家对同一村民有**独立好感**；高好感可跟随协助除草，低好感可能破坏玩家附近耕地。 |
| **天气与时间** | 昼夜循环、季节、雨雪等天气表现与模拟配合。 |
| **联机** | 菜单中可 **本地单机**、**做主机** 或 **加入局域网**；加入方会接收世界快照后与主机同步。 |
| **编辑（主机）** | 生物群系笔刷、气候编辑、农耕工具等（具体权限以游戏内提示为准）。 |

小地图支持多种视图（生物群系、海拔、湿度等），按 **V** 切换；地图上以不同颜色区分本地玩家、其他玩家与村民。

## 默认操作（键盘）

可在 Godot **项目 → 项目设置 → 输入映射** 中修改；游戏启动时也会尝试注册以下动作：

| 动作 | 默认按键 |
|------|-----------|
| 移动 | **W A S D** |
| 与附近村民互动 | **E** |
| 切换小地图图层 | **V** |
| 开关笔刷模式 | **B** |
| 笔刷修饰 | **Q** |
| 笔刷半径 - / + | **[** **]** |
| 开关演替加速 | **F** |
| 演替倍率 - / + | **PageDown** / **PageUp** |
| 开关农耕模式 | **G** |
| 切换农具 | **R** |
| 笔刷/部分工具应用 | **鼠标左键**（勿点在 UI 上） |

## 工程结构（简要）

```
Scenes/          # 主场景、世界、UI、玩家等 .tscn
Scripts/
  Core/          # GameManager、常量等核心逻辑
  World/         # 世界数据、地块、小地图
  Ecology/       # 生态模拟、动物、村民、农耕相关
  Network/       # 联机与同步
  Player/        # 玩家控制
  UI/            # 菜单与 HUD
  Weather/       # 昼夜、季节、天气
Shaders/         # 着色器资源
```

## 显示

默认窗口分辨率在 `project.godot` 的 `[display]` 中配置；玩家摄像机缩放可在 `Scenes/Player/Player.tscn` 的 `Camera2D.zoom` 调整。

---

**提示**：联机时请保证防火墙放行所选 **UDP 端口**（代码中默认端口见 `Scripts/Core/Constants.cs` 的 `DefaultPort`，一般为 **7777**）；主机与客户端需使用兼容的 Godot / .NET 版本构建。
